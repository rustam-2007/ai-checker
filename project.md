# AI Checker

AI Checker is a simple web app for checking text and estimating whether it may have been generated by AI.

## MVP

The first version should support:

- A web page where users paste text.
- An API endpoint that accepts text for checking.
- A clear result response that can later include score, label, and explanation.

No detection logic is implemented in the initial setup.

## Future Phases

- Add YouTube video support by extracting or accepting transcript text.
- Add authentication, saved checks, or billing only if needed.
- Build a Chrome extension after the Web MVP and API are stable.

## Product Principles

- Start small.
- Make the main text-checking flow fast and obvious.
- Prefer reliable, explainable outputs over complex UI.
- Keep the system easy to replace or improve as the detection approach evolves.
