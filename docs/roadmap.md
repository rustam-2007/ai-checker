# Roadmap

## Phase 1: Project Setup

- Create project documentation.
- Create web, API, shared, docs, and skills folders.
- Keep app folders as placeholders.

## Phase 2: Web MVP Scaffold

- Generate a minimal Next.js app in `apps/web`.
- Add a single text input flow.
- Add basic styling.

## Phase 3: API MVP Scaffold

- Generate a minimal NestJS app in `apps/api`.
- Add a text-check endpoint.
- Add validation for empty or invalid input.

## Phase 4: Shared Contracts

- Add shared TypeScript request and response types.
- Use the shared types from both web and API.

## Phase 5: Detection Logic

- Add first simple AI-checking implementation.
- Return a score, label, and short explanation.

## Phase 6: YouTube Support

- Add transcript input or transcript extraction.
- Reuse the same text-checking API flow.

## Later: Chrome Extension

- Build only after the Web MVP and API are stable.
